# Scripts

### Install dependencies

```bash
yarn install
```

## Following commands should be run from monorepo root directory:

### Deploy factory

```bash
yarn deploy-factory
```

### Deploy linkdrop mastercopy

```bash
yarn deploy-mastercopy
```

### Generate links

```bash
yarn generate-links
```

### Setup linkdrop

Deploys factory, linkdrop mastercopy and generates links

```bash
yarn setup-linkdrop
```

### Run server

```bash
yarn server
```

### Claim tokens

```bash
yarn claim
```
