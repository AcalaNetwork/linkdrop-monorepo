# Scripts

### Install dependencies

```bash
yarn install
```

### Following commands should be run from monorepo root directory:

### Compile contracts to get metadata

```bash
 yarn compile-contracts
```

### Deploy ERC20 Token

```bash
yarn deploy-erc20
```

### Deploy factory

```bash
yarn deploy-factory
```

### Deploy linkdrop mastercopy

```bash
yarn deploy-mastercopy
```

### Generate links

```bash
yarn generate-links
```

### Setup linkdrop (deploys factory, linkdrop mastercopy and generates links)

```bash
yarn setup-linkdrop
```

### Run server

```bash
yarn server
```

### Claim tokens

```bash
yarn claim
```
